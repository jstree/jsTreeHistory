개발 프로세스

바로보드는 다양한 개발 도구를 사용하여 효율적으로 요구사항 및 태스크를 점검하며
빌드, 소스관리, 모니터링에 있어서도 품질관리에 이상이 없도록 최적화된 프로세스를 진행하고 있습니다.

개발 프로세스 흐름도

1. 요구사항 확인 및 일감 분배 단계

Confluence로 요구사항을 등록하고 프로젝트 관련 산출물 및 지식을 관리합니다.

JIRA를 통해 업무(issue)를 등록합니다. 세부적으로 기본 Task 및 Sub Task 등록 가능합니다.

  • Confluence
    요구사항 등록, 산출물 관리, 지식 공유
  • Jira
    업무 등록 및 관리

2. 개발 단계

EclipseMylyn을 통해 개발 완료한 내용을 JIRA에 알려주며 FishEye와 Crucible을 통해 형상관리 서버(SVN 또는 GitLab)와 연동하여 프로젝트 소스의 변경사항을 관리합니다.

  • Mylyn
    Eclipse로 등록된 일감을 가져옵니다.
  • FishEye
    형상관리 서버(SVN 또는 GitLab)와 연동하여 프로젝트 소스에 대한 변경사항을 관리

3. 빌드 단계

빌드 툴들의 도움을 받아 빌드를 수행합니다.

  • Apache Maven
    Java프로젝트의 빌드, 테스트, 배포, 버전관리를 자동으로 도와줍니다.
  • Nexus
    가장 널리 사용되는 Maven Repository로서 라이브러리 관리를 도와줍니다.
  • Hudson
    메이븐에서의 빌드 스크립트를 추출해 빌드를 주기적으로 런칭시키는 빌드 서포트 툴입니다.
  • Sonar
    Hudson의 빌드 결과를 등록하여 코드 품질관리를 수행합니다.

4. 모니터링 단계

4-1. 서버모니터

  • Server Daemon Moniter
    서버들의 리스트가 올라와 있는지, 정해진 포트에서 잘 실행되고 있는지 여부를 체크해 줍니다.
  • Server Daemon Moniter
    (probe-2.3.2)
    웹상에서 Request, Sessions, JSP, Data Source, Logs, Threads, Connectors, Clustors,JVM, Java Service Wrapper, System 등의 사항들을 관리를 도와주는 툴입니다.

4-2. 서버로그 분석

  • 구글 웹로그 분석
    (Google Analytics)
    방문자의 사이트 활동, 사이트 유입경로, 고객의 재방문을 유도하는 방법에 대한 최신 분석 자료 등을 제공합니다.
  • Deep Log Analyzer
    방문자의 유입 흐름과 행동들을 분석하고 트래픽, 검색어, 검색 엔진, 사용 브라우저, OS등 각종 통계를 제공합니니다.
  • Alterwind Log Analyzer
    사이트 방문자 통계, 검색 엔진, 검색어 분석 및 트래픽 분석, 웹사이트 에러 찾기 등의 기능을 수행하는 툴입니다.

4-3. 네트워크 관리

  • 네트워크 모니터링 SNMP
    (mrtg-2.16.4)
    어플리케이션, 가상 서버, 멀티사이트 등의 네트워크 모니터링을 수행합니다.
  • Switch Router
    (D-LINK DIR-825)
    유무선 공유기인 D-LINK(DIR-825)의 네트워크 관리 툴입니다.
  • DDNS
    (noti)
    유동 IP 를 전용선에 부여되는 고정 IP 처럼 사용할 수 있도록 해주는 시스템입니다.

4-4. DB 관리

  • ORACLE 11g EM
    (EM 1.0)
    ORACLE 데이터베이스 관리 툴입니다.
  • HADOOP(준비중)
    Hadoop 데이터베이스 관리 툴입니다.
  • MySQL
    (phpMyAdmin)
    MySQL 데이터베이스 관리 툴입니다.

4-5. 보안 관리

  • SSO CAS Jasig
    (Server)
    하나의 인증 장치로 다중 어플리케이션 접근을 허용하는 인증 시스템입니다. (서버).
  • SSO CAS Jasig
    (Client)
    하나의 인증 장치로 다중 어플리케이션 접근을 허용하는 인증 시스템입니다. (클라이언트).

5. 소스 관리

5-1. 코드 관리

  • SVN (Polarion)
    여러명이 동시에 개발 프로젝트를 진행할 시 소스관리를 할 수 있게 하나의 서버에서 관리할수 있도록 도와주는 툴입니다.
  • GITLAB
    여러명이 동일한 프로젝트를 진행 할 때 쉽게 소스관리를 할 수 있도록 도와주는 툴입니다. SVN과 비슷하나 분산환경에서 사용하기 용이합니다.

5-2. 서버 솔루션

  • DMS
    (OpenKM)
    문서(산출물) 관리 시스템입니다.
  • NAS
    (Xpenology DSM 4.2)
    네트워크에 접속해서 백업, 파일공유등의 활동이 가능한 대용량 저장장치 입니다.
  • Web FTP
    (net2ftp)
    웹상에서 FTP 기능(파일 전송)을 수행하도록 도와주는 툴입니다.
  • Web Hard
    (ajaxplorer)
    Apache 환경에서 동작한는 설치형 웹하드 시스템입니다.
  • eXtplorer
    (eXtplorer-2.1.0RC3)
    웹 기반 파일 관리 시스템 입니다.

5-3. 클라우드 솔루션

  • Tonido
    설치형 클라우드 서비스 시스템입니다.
